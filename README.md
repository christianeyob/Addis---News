# Addis News - Frontend

Next.js app with Prisma, Neon, shadcn/ui, Framer Motion.

## Tech
- Next.js 14
- shadcn/ui + Tailwind
- Framer Motion animations
- Prisma ORM + Neon Postgres
- Vercel deployment

## Features
- News dashboard 
- Responsive design
- Smooth animations
- TypeScript end-to-end

## Setup
npm install
create .env in the root folder and add your database url
npm run dev
